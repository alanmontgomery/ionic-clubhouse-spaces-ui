### Included in this Ionic React Template/UI
* Floating tab bar
* Custom cards/colors
* Pullstate
* Card Modals

### To run

```javascript
npm install
ionic serve
```

Alternatively, you can add the iOS, Android platform and run natively. Keep in mind though, the Native scanner will only work on a physical device.
